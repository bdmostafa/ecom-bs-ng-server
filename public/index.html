API Endpoints

Users

  • POST /api/users/create create a new account
  • POST /api/users/login login as user
  • GET /api/users get all users
  • PATCH /api/users/update/me update an user by the same user
  • DELETE /api/users/delete/me delete an user by the same user
  • DELETE /api/users/delete delete an user by super admin

Products

  • GET /api/products get all products
  • GET /api/products/{productId} get a product by id
  • GET /api/products/category/{categoryName} get products by category
  • POST /api/products/create create a new product
  • PATCH /api/products/update/{productId} update a product
  • DELETE /api/products/delete/{productId} Delete a product
  • GET /api/products/generate-products Generate product from third party API

Orders

  • GET /api/orders get all orders
  • GET /api/order/{orderId} get an order by ID
  • GET /api/orders/pending-orders get pending orders
  • POST /api/orders/create create a new order
  • PATCH /api/orders/update/{orderId} update an order
  • GET /api/orders/{orderId} get a single order
  • GET /api/orders/orders-by-date get orders by date
