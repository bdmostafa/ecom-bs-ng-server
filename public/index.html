API Endpoints

Users

  • GET /api/users Get all users (authorization for admin and super admin)
  • GET /api/users/me Get loggedIn user info (Authentication for loggedIn user)
  • POST /api/users/create Create a new account/ Register an user
  • POST /api/users/login Login as an user/ customer
  • PUT /api/users/update/me Update loggedIn user info (authorization for loggedIn user)
  • PUT /api/users/update/{userId} Update any user info within role (authorization for only super admin)
  • DELETE /api/users/delete/me Delete an user (authentication for loggedIn user)
  • DELETE /api/users/delete/{userId} Delete any user (authorization for only super admin)
  • DELETE /api/users/logout Logout an user (Authentication for loggedIn user)

Products

  • GET /api/products Get all products
  • GET /api/products/{productId} Get product detail by id
  • GET /api/products/category/{categoryName} Get products by category
  • GET /api/products/generate-products Generate products from third party API (authentication for admin and super admin)
  • POST /api/products/create Create a new product (authentication for admin and super admin)
  • PUT /api/products/update/{productId} Update a product (authentication for only super admin)
  • DELETE /api/products/delete/{productId} Delete a product (authentication for only super admin)

Orders

  • GET /api/orders get all orders (authorization for only super admin)
  • GET /api/orders/{orderId} Get a single order by id (authorization for only super admin)
  • GET /api/orders/orders-by-date/{date} Get orders per day (authorization for only super admin)
  • GET /api/orders/pending-orders Get pending orders (authorization for admin and super admin)
  • GET /api/orders/user/my-orders Get loggedIn user's orders (authentication for loggedIn user)
  • POST /api/orders/payment Payment for an order (authentication for loggedIn user)
  • POST /api/orders/create Place an order (authentication for loggedIn user)
  • PUT /api/orders/update/{orderId} Update any order status (authorization for admin and super admin)
